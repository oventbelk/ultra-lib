# ultra-lib
Micro ultra-lib tool

## Features
- Lightweight CLI tool
- Easy to use

## Installation
Use composer to install:
```bash
composer require your-vendor/ultra-lib
```

## Usage
```php
require 'vendor/autoload.php';

use YourVendor\UltraLib;

$tool = new UltraLib();
$tool->execute();
```
